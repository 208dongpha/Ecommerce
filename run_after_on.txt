3. Kết nối Trino với Delta Lake

Bạn đã có catalog:

SHOW CATALOGS;

trả:

delta
system

=> Trino đã nhận Delta Lake.

4. Tạo schema trong Trino

Vào Trino:

docker exec -it trino trino

Tạo schema:

CREATE SCHEMA IF NOT EXISTS delta.ecommerce
WITH (
 location = 's3://ecommerce/'
);
5. Register Gold Table

Ví dụ Gold nằm ở:

s3://ecommerce/gold/sales_metrics

thì:

CALL delta.system.register_table(
 schema_name => 'ecommerce',
 table_name => 'sales_metrics',
 table_location => 's3://ecommerce/gold/sales_metrics'
);
6. Kiểm tra
SHOW TABLES FROM delta.ecommerce;

phải thấy:

sales_metrics

Đọc thử:

SELECT * 
FROM delta.ecommerce.sales_metrics
LIMIT 10;
